# CourseApp

App to manage courses and students
 - Students CRUD
 - Classes CRUD
 - Assign students to class
 - Search students/classes by any field 

#### Tools 
- Java 8
- Spring boot 1.5.9.RELEASE
- Apache Maven 3.3.9

#### Build and execute
To build project(generate jar file)
```
mvn clean install 
```

To excecute
```
java -jar target/demo-0.0.1-SNAPSHOT.jar
```
